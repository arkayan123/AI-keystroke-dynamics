# AI Keystroke Dynamics for Continuous Authentication

## Overview

AI Keystroke Dynamics for Continuous Authentication is a Flask-based behavioral biometric authentication system that continuously verifies user identity using typing patterns.

Unlike traditional password authentication, the system analyzes typing behavior such as dwell time, flight time, typing speed, and typing rhythm to identify legitimate users and detect unauthorized access.

## Features

- Real-time keystroke capture
- Continuous authentication
- Behavioral biometric analysis
- Machine Learning based user verification
- Flask web dashboard
- Feature extraction and preprocessing
- User profile management
- Security monitoring

## System Architecture

User Input
↓
Keystroke Capture
↓
Feature Extraction
↓
Machine Learning Model
↓
Authentication Decision
↓
Allow / Reject Access
